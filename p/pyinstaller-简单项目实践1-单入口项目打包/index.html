PyInstaller 简单项目实践1-单入口项目打包

使用PyInstaller进行Python简单的单入口项目打包实践。

2025-02-16
阅读时长: 1 分钟

PyInstaller 支持跨平台打包，可以将 Python 代码打包成适用于 Windows、macOS 和 Linux 系统的可执行文件。它能够自动查找程序所依赖的 Python 库文件，并将其打包到一个文件或者一个文件夹中。

1.安装 PyInstaller

使用 pip 安装最新版 PyInstaller：

1

pip install pyinstaller

升级已安装的 PyInstaller：

1

pip install --upgrade pyinstaller

在 Windows 系统上，PyInstaller 会自动安装依赖的 PyWin32 或 pypiwin32。

2.基本打包方法

打开命令提示符或终端，定位到项目文件夹 “E:\project\script.py”

打包为单个文件夹（默认）

1

pyinstaller script.py

这会在当前目录下生成：

  • build/ 文件夹：包含临时文件，可安全删除
  • dist/ 文件夹：包含可执行文件和所有依赖项

打包为单个可执行文件

1

pyinstaller -F script.py

或

1

pyinstaller --onefile script.py

这样打包后，在 “dist” 文件夹下会直接生成一个单独的可执行文件，例如 “script.exe”,双击它就可以运行项目，就像运行普通软件一样。这种方式生成的文件体积可能会更大，并且启动时间可能稍长，因为它需要在运行时解压所有依赖。

指定生成文件的路径
果不想让 PyInstaller 在默认的 “dist” 文件夹下生成文件，可以指定输出路径。

1

pyinstaller --distpath=output_folder script.py
